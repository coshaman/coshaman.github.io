# Portfolio Design Direction

## Theme

Quiet academic portfolio with editorial contrast. The site should feel personal, credible, and polished rather than flashy.

## Visual Principles

- Use warm paper backgrounds with white content surfaces.
- Pair deep teal for structure with a small amount of brick orange for emphasis.
- Keep motion minimal; rely on layout, spacing, borders, and imagery instead of animation.
- Use the existing profile photo and project screenshots as the main visual assets.
- Make the first viewport immediately identify the person, major, profile image, and primary actions.

## Layout

- Sticky navigation links to all required sections.
- Hero uses a two-column layout on desktop and a single-column layout on mobile.
- Sections use clear headings, compact descriptive text, and cards only for repeated content.
- Resume should include an obvious CV download link.
- Projects should show screenshots, categories, descriptions, and external links.

## Accessibility

- Maintain semantic headings and section landmarks.
- Keep color contrast readable.
- Use descriptive alt text for images.
- Preserve keyboard focus styles and a skip link.
- Ensure no horizontal overflow on mobile.

## Content Requirements

- About Me: name, major, short self-introduction.
- Profile photo or avatar.
- Contact information.
- CV / Resume section with education, skills, and experience.
- At least two activities or projects.
- Navigation menu linking to each section.
- Planned courses: algorithms, system programming, operating systems.
- AI Usage section with tool name and one prompt.
